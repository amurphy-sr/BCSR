Meta Data

page props for title, description, keywords, and "Meta Robots"

  • if toggled off then <meta name="robots" content="noindex, nofollow"/>
  • else no tag.

<meta name="canonical" is searched for in page <headcode>

  • if found, eg. output is <meta name="canonical" content="http://link/to/different/page.php" />
  • else XSL outputs content="http://link/to/current/page.php"

Open Graph accessible via page via properties, og:title, and og:url.

  • <meta property="og:title" content="Current page title | Bard College at Simon's Rock" />
  • auto created from "full" page title
  • OG URL is empty by default
    • this outputs current page URL, eg. <meta property="og:url" content="http://link/to/current/page.php" />
    • if filled that value is used, eg. content="http://link/to/different/page.php"
  • OG Image is filled with default value "/image/og.png"
    • outputs <meta property="og:image" content="/image/og.png" />
    • user can change value to output content="/image/different-og.png"
    • if no value XSL will write content="/image/og.png".

<a id="video">Video</a>

Using the "Insert/Edit Video" button in the WYSIWYG gives you the otpion to insert 2 sources, a poster image and dimensions.

in standard editable regions you only need to provide one source value, a poster image, and dimensions

Hero Parameters

Each page with a banner image/video option selects this via page props, this gives access to image or video editable regions. Each region is filled with a snippet, filled with default content. Each snippet can be extended to allow for a modal pop-up. This is accomplished by adding a button to your current snippet and giving the button's (which is just an <a/> tag) href attribute the value #, and adding a modal snippet below the current snippet.

Transformation Snippets

Hero Video

Ability to add video content to the Hero Video Editable Region of a template. Snippet takes in the title, content, buttons and video source. Use the hero buttons snippet for adding buttons to a hero video. This snippet will appear by default in the Hero Video Editable Region. The class to build the background image node for CSS will be constructed from the title.

Hosted Video

Hosted video can be added in the WYSIWYG add video option. All you need to do is choose one of the video types, and the remaining will be added using xsl. The file names and locations must be the same in order for this to work correctly. In addition to the video you can add a poster/fallback image for the video in the video dialog box.

iFrame Video

iFrame video can be added in the WYSIWYG add video option. Use the embed tab and paste in the <iframe> embed code obtained from Youtube for it to display properly.

Hero Image

Ability to add an image to the Hero Image Editable Region of a template. Snippet takes the title, content, buttons, a main image, and a large image. Use the hero buttons snippet for adding buttons to a hero image. The snippet will appear by default in the Hero Image Editable Region. The class to build the background image node for CSS will be constructed from the title and will be added to the <head> of the page.

Hero Modal

Ability to have a button open a video modal, instead of navigating to another page. Snippet takes in the title, video source (hosted, or iFrame), and the modal call to action button.

Hero Buttons

Ability to add buttons to the hero video or image. Snippet takes in button text and link. If you wish for the button to open a modal set the link to be “#” and ensure that you have included the modal snippet in the same hero editable region. Adding a new table row will create multiple buttons, so if you want three buttons create three table rows in one button snippet. Buttons created with this snippet will be surrounded with <div class="hero-buttons">

Center Buttons

Ability to add buttons in an editable region. Snippet takes in button text and link. If you wish for the button to open a modal set the link to be “#” and ensure that you have included the modal snippet in the same editable region. Adding a new table row will create multiple buttons, so if you want three buttons create three table rows in one button snippet. No matter how many buttons you add using this snippet they will all be surrounded with <div class="text-center">.

Basic Button

Ability to create a basic button. Snippet takes in the button class and button text and link. This is a single button and will not build multiple buttons in a row. For this functionality please use the Center Buttons snippet.

Info Boxes

Ability to create colored info boxes as seen on the campus-landing page. Snippet takes in the class and the content. The color of the box is determined by the css. Only the first three rows of the snippet will display.

Panel/Tab

Ability to create the tabbed panel as seen on the campus-landing page. Snippet takes in the tab title, tab content, which could be an image and caption or HTML content via a source code asset or through WYSIWYG source code editor. If adding an image and a caption make sure to use the styles dropdown to add the accordion image class to the image, and the accordion text class the caption. Adding a new row to the snippet will add a new tab title and tab content region to the panel/tab.

Map

Unique to the map page. By default will be part of the Locations Editable Region on the map page. Snippet will take in the location name, latitude and longitude, z-index, image, description and link. Adding a new row will allow you to add a new location to the map. Removing a row will remove the location from the map.

Image List Selection

Ability to add the image list selection as seen on the gateway page. Snippet takes in the title, image, content and link. The content section is prepopulated with another two-column snippet that builds the two-column layout that is seen on the gateway page. To create the bulleted list you can either use the WYSIWYG editor or insert the pre-styled arrow list snippet.

Circle

Ability to add the circles as seen on the giving landing page. Snippet takes in the circle class color, the title and link. Only the first four circles in the snippet will display.

Thumbs

Ability to add the thumbs as seen on the giving landing page. Snippet takes in the section heading, image, link and caption. Only the first three items of the snippet will display.

Action Items

Ability to create the action items bar as seen on the master section landing page. Snippet takes in the icon class, text and link. Adding a new row in the snippet will add space for a new action item.

Profile Thumb

Ability to add thumbnail images as seen on the profile page. Snippet takes in the image, title and additional text. All of the text classes are available in the styles dropdown menu. This is meant to be used within a Basic Layout snippet.

Profile Project Image

Ability to build a project image group as seen on the profile page. Snippet takes in two images. The content below the images is placed outside of the snippet. Adding a new row to the snippet will not add aditional content to the page.

Image Carousel

Ability to build the rotating carousel as seen on the student life page. Snippet takes in the image, text and link. The class for the image is built by xsl using the title. Adding additional rows to the snippet will add additional items to the carousel. Each of the background images that are added will be written to the <head> of the page.

Top Left Callout

Ability to build the top left callout region as seen on the profile page. Snippet takes in the color class and the text you wish to display. Can also include a link if desired.

Top Right Callout

Ability to build the top right callout region as seen on the profile page. Snippet takes in the color class and the text you wish to display. Can also include a link if desired.

Bottom Left Callout

Ability to build the bottom left callout region as seen on the profile page. Snippet takes in the color class and the text you wish to display. Can also include a link if desired.

Advanced Layout

Ability to build an advanced div layout structure in the WYSIWYG editor. Snippet takes in the class(es) for the "row" div. It takes in the "column" class(es) and it takes in the column content. Creating a new row will allow you to add multiple columns, however the columns make use of the Bootstrap grid, so all of the column classes must add up to 12. Any new rows will be added inside of the main "row" div.

Basic Layout

Ability to create a similar layout to the Advanced, however you cannot edit the class on the main "row" div using this snippet. Snippet takes in the column class and content. Creating a new row will allow you to add multiple columns, however the columns make use of the Bootstrap grid, so all of the column classes must add up to 12.

Div Only Layout

Ability to create a div structure with a class and add content within that div. Snippet takes in the div class and the content for that div. Unlike the Advanced and Basic layouts this snippet will only create a single div with content, adding a new row in this snippet will not do anything for the layout.

Non-Transformation Snippets

Ordered Circle List

Templates / Page Types

Alert

The alert is a pcf file located in /_resources/includes/alert.pcf, when published it is published as an include file to the production, here. This include file is included on all pages of the site at all times. However the only time that the alert displays is when you toggle the page parameter for the alert.pcf file to display the alert. In addition to the show/hide toggle there is also a toggle to choose the level of alert, info, warning or danger. Each of these types has its own editable region where you can build out the alert message using the wysiwyg and the styles dropdown for the link class. The special character that is part of the "Learn More" link can be added with the special character component of the wysiwyg.

See also <a href="#alert">Alert style drop-down</a>
